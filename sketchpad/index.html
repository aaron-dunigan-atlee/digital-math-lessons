Here are some sketches I have made with “The Geometer’s Sketchpad.”  Some were made for classes I teach; others just for my own entertainment or education.  As a result, the amount of explanation varies; if you can’t figure out what a sketch is for or how it works, choose “Show All Hidden” from the Display menu, look at the Script View of custom tools, or check the Properties of the various objects in the sketch—or send me an email.

 

Full-blown Sketches: Sketches created to demonstrate an idea in class or allow students to explore.

Hypercubes

I made this sketch to use for a unit in which I teach seniors about four-dimensional geometry.  I used it as a visual aid as we tried to visualize hypercubes by thinking about how a two-dimensional “Flatlander” might try to visualize three-dimensional cubes.

Normal Curve

A sketch for calculating and visualizing percents under the normal curve.  Also includes a “Z-value Calculator.”

Inequalities

This sketch uses dynagraphs to show how various operations affect an inequality.  Useful for demonstrating that you must change the inequality sign when multiplying or dividing both sides of an inequality by a negative number.

Complex Dynagraph

A dynagraph for complex numbers.  Input and output values are both plotted on a complex plane, and can be traced to see how an “input path” corresponds to an “output path” for a given complex function.

Taxicab Geometry

This interesting non-Euclidean geometry was invented by Hermann Minkowski.  I’ve seen some other Sketchpad representations, but their loci are continuous.  The original Taxicab Geometry was conceived as a discrete set of points on the integer lattice—which is what I’ve done here.  (Sketch in progress—I still need to add a hyperbola tool and some explanations.)

Modular Arithmetic

A visualization of arithmetic in a modular number system.  Calculates sums, products, and inverses of numbers, and plots them on a circular number line.

Points in an Interval

This sketch gives a visual “proof” of the fact that there are as many (real) numbers between 0 and 1 as there are on the whole number line; or, equivalently, that there are as many points on a line segment as there are on an infinitely long line.

Lights Puzzle

A puzzle in which you try to light up an array of lights by pushing buttons that have various effects on the lights.  Includes a template for creating your own puzzles.

Turtle Race

Inspired by the “Logo” program (in which you program a turtle to create shapes by giving him walking instructions), I created this sketch for a colleague whose middle school students were exploring tables and graphs.

Spirograph

Like the old toy—only more high tech!

Spirolaterals

A spirolateral is a kind of line design based on a list of lengths and an angle.

Polygons with Fixed Perimeter

A common exercise is to ask students to examine the possible areas of a rectangle with fixed perimeter.  The classic example is that Farmer Arepo has a fixed amount of fence and he wants to maximize the area of his rectangular sheep pen.  Such a rectangle is fairly easy to construct in Sketchpad.  In this sketch, I attempted to extend the idea to more general polygons.  The result is not as elegant as I wanted it to be, but it at least works. 

Higher-Order Parabolas

What happens when the directrix of a parabola is not a line, but another parabola?  This sketch shows the answer.  (See also “Generalized Parabolas” in the next section.)

Parallelity Checker

How do you know if two line segments are parallel?  Here are six different ways to check!

Solids of Revolution

A sketch to visualize a “Solid of Revolution” generated by revolving a function about the x-axis.  At this point, the sketch only animates the revolution—it does not estimate the volume of the solid (I’m thinking about how to go about this.)

 

 

 

Custom Tools and Toolkits: Custom tools to perform some common (and some not-so-common) tasks.

Combinatorics and Probability

Tools for calculating factorials, permutations, and combinations (these tools use the iteration workaround, described below); also a random number generator.

Complex Arithmetic

Some tools for doing complex arithmetic: given points A and B on a coordinate system (interpreted as the complex plane), I made tools to calculate and plot A+B, A*B, and 1/A.  (See also the Complex Dynagraph, above.)

Flow Charts

My wife needed to make a flow chart for a class she was taking.  After she made a frustrating attempt at making them with MS Word, I realized the solution: Sketchpad!  These simple custom tools create arrows, rectangles, diamonds, and triangles, and then you can add text labels.  Voila!

Number Line

You can easily make a number line by defining a coordinate system and then hiding the y-axis.  But there won’t be a tick-mark or label for the number zero.  This tool makes a number line that includes the missing label and tick-mark.

Regular Polygons as Loci

This interesting tool constructs a regular polygon, but one that is defined as a locus rather than a set of segments.  Why would you want this?  Two reasons: 1) The number of sides can be changed dynamically depending on a parameter or the result of a calculation;  2) The locus is all one path object, so you can construct a point on the polygon and drag or animate it around the whole perimeter.

Number Theory

Tools for calculating GCD, LCM, a mod n, and modular inverses; testing for prime numbers; counting factors of n; and more.

Generalized Parabolas

This sketch includes a custom tool for making a "Parabola with Non-Linear Directrix"—that is, the locus of points that are equidistant to a curve and a point.  (See also “Higher-Order Parabolas” in the section above.)

Tips and Tricks: Some details and advanced tricks to make your Sketchpad life easier.

Table with Constant Increment

How to create a table so the independent variable is incremented by a constant value.

 

Iteration Workaround

Officially, you can only iterate points or parameters that are independent: that have no parents.  But you can actually iterate dependent objects if you first construct them as independent, then construct the iteration, then re-define the objects to depend on other objects.  More details in the sketch.

Conditionals and Objects that Sometimes Exist

There is no explicit “if-then” structure in Sketchpad, but you can construct its equivalent.  For example, suppose you have a slider whose value is called a, and you want to see a red square if a > 0, or a blue triangle if a < 0.  This sketch shows you how.

 

Selecting a Non-existent Object

Your constructed object doesn’t exist?  No problem!  You can still select it and set its properties.  (This is about objects that don’t exist, not ones that are simply hidden.  See the topic above for why you would want to construct a non-existent object.)

 

Discrete Loci

Working with a discrete locus.

 

Discrete Loci—Advanced

How to create a locus defined on a specific discrete domain (integers, multiples of 2, powers of 7, etc.).

Investigations: These are activities I developed for my high school geometry/algebra classes.

 

Slopes of Parallel and Perpendicular Lines

A short investigation to discover that parallel lines have equal slope and that perpendicular lines’ slopes are opposite and reciprocal (based on pp. 165-166 of Discovering Geometry).

 

Compass-and-Straightedge Constructions with the Geometer’s Sketchpad

Students had practiced some basic construction techniques using physical compass and straightedge—now I asked them to do the same constructions with Sketchpad, using just the compass and straightedge tools.  This allowed the students to get a “feel” for Sketchpad’s interface.
